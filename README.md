## 📌 What this app does

hk-dashboard is a desktop app for viewing live Hong Kong city data in one place. It brings together transport, weather, and public data in a clear dashboard. It is made for everyday users who want quick access to useful city info without needing to search through many sites.

## 🧭 How to use the dashboard

### View city data

The main screen shows live city information in simple panels. You can check current conditions, recent updates, and useful public data at a glance.

### Check transport

Use the transport view to see bus ETA and transit updates. This helps when you want to plan a trip or check service times before leaving home.

### Check weather

Use the weather section to view current weather details. It can help you decide what to wear or whether to carry an umbrella.

### Explore the map

The app uses map tools to show location-based data. You can zoom in, move around, and inspect different parts of Hong Kong.

### Switch between pages

The dashboard includes 15 pages. Use the menu or tabs in the app to move between views. Each page focuses on a different kind of city data.

## 🌐 Data source

## 🧩 Common issues

### The app does not open

- Make sure the download finished
- Check that you opened the right file
- Try running it again as the same user who downloaded it

### Windows says the app is blocked

- Click More info
- Choose Run anyway if you trust the source
- This can happen with new apps from GitHub

### The data does not load

- Check your internet connection
- Wait a few seconds and refresh the app
- Try again later if the public data feed is busy

### The screen looks too small

- Maximize the window
- Use Windows display scaling
- Reduce zoom in the browser if you are viewing it through a web mode
